Usage Of Application

  • User should enter their Registration Number as user name to login the system (Ex: If Reg. No. is E/01/001, user name for system is 01001).
  • Only admins can add course to the system
  • In the course registration if there are no projects, assignments or Mid or End exams for particular course, put 0 in respective fields.
  • In EXPECTED GPA and COURSE REGISTRATION first course should be selected from draw down menu and then OK button should be pressed for select that particular course for sub-sequent actions.

 

Usage of Buttons

  • ENTER MARKS – User can enter their marks for pre-registered courses.
  • CURRENT GRADE - User can view their current GPA and rank
  • EXPECTED GRADE – User can calculate expected GPA for next semesters to achieved preferred GPA.
  • REQUIRED MARKS – User can calculate required marks for selected subject for get required marks.
  • REGISTER COURSE – Register courses to the system (Only admins have this option)
